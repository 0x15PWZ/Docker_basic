🧪 Practice Exercises
_____________________
✅ 1. Docker CLI သုံးပြီးအောက်ဖော်ပြပါလုပ်ဆောင်ပါ
nginx container တစ်ခု run ပြီး browser မှာ verify
mysql container တစ်ခု run with root password env variable
Docker image တစ်ခုကို pull & inspect

✅ 2. Dockerfile Practice
Custom Python script app တစ်ခုရေးပြီး Dockerfile တည်ဆောက်ပါ
Web app တစ်ခုကို Dockerfile နဲ့ build → run → verify

✅ 3. Docker Compose Practice
2-container App: Flask + Redis
3-container App: Django + PostgreSQL + Nginx (Reverse Proxy)

✅ 4. Docker Hub Practice
Docker Hub account တစ်ခုဖွင့်ပြီး
ကိုယ့်ရဲ့ image တစ်ခုကို build & push
docker login
docker tag yourimage username/yourimage:tag
docker push username/yourimage:tag
